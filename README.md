## 贸易跟单系统

### 技术栈

1、 Vue v2.6.x
2、 Node v12.x

### 项目运行
1、 使用npm install 
2、 使用npm run dev 开发运行
3、 发布npm run build 进行项目打包

### 项目结构

|—— trade-app
  |—— public
    |—— index.html          //项目文件入口页面
    |—— assets              //静态资源
  |—— src                   //开发目录
    |—— main.js             //项目主入口文件
    |—— pages               //整个项目页面组件包
